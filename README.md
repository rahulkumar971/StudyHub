StudyHub is a full-stack web application that helps students access all their study materials in one place. It provides a centralized platform to organize notes, videos, PDFs, and useful links by subject and topic, making learning more structured and efficient.

🚀 Features
📚 Centralized study materials (Notes, PDFs, Videos, Links)
🔍 Search and filter resources quickly
👤 User authentication and secure login
📁 Subject-wise and topic-wise organization
⬆️ Upload and manage study resources
📱 Responsive design for desktop and mobile
⭐ Bookmark important materials for quick access

🎯 Purpose
StudyHub is designed to solve a common problem faced by students: study materials are often scattered across different platforms. This application brings everything together into a single, easy-to-use dashboard where students can find and manage resources efficiently.
